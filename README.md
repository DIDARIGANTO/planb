# 2ГИС · Панель планирования оплат РГП

Личная воронка продлений: план, прогноз закрытия, бонус 5%, трекер оплат.

---

## Как опубликовать сайт (через сайты, без командной строки)

### Шаг 1. Создать репозиторий на GitHub
1. Зайди на https://github.com → кнопка **New** (зелёная, слева вверху).
2. Repository name: например `rgp-panel`.
3. Оставь **Public** (или Private — без разницы).
4. Нажми **Create repository**.

### Шаг 2. Залить файлы проекта
1. На странице нового репозитория нажми ссылку **uploading an existing file**
   (или кнопка **Add file → Upload files**).
2. Перетащи в окно браузера **всё содержимое этой папки**:
   - папку `src`
   - `index.html`
   - `package.json`
   - `package-lock.json`
   - `vite.config.js`
   - `.gitignore`

   ⚠️ Папки `node_modules` и `dist` загружать НЕ надо — они создадутся сами.
3. Внизу нажми **Commit changes**.

### Шаг 3. Опубликовать на Vercel
1. Зайди на https://vercel.com → войди через **Continue with GitHub**.
2. Нажми **Add New… → Project**.
3. Найди свой репозиторий `rgp-panel` → **Import**.
4. Vercel сам поймёт, что это Vite. Ничего менять не нужно — просто нажми **Deploy**.
5. Через ~1 минуту получишь ссылку вида `rgp-panel.vercel.app`.

Готово. Открывай ссылку с телефона или компьютера, делись с кем нужно.

## Что важно знать

- **Данные хранятся в браузере** (localStorage). Это значит: у каждого устройства/браузера
  свои данные, они не улетают на сервер. Если откроешь сайт на другом компьютере —
  там будет пусто. Это нормально для личного инструмента.
- Хочешь, чтобы данные были общими и хранились в облаке (база данных) — это следующий
  шаг, нужна доработка. Спроси, помогу.
- Любая правка кода: меняешь файл прямо на GitHub → Vercel пересоберёт сайт сам за минуту.

## Запустить у себя на компьютере (по желанию)
```
npm install
npm run dev
```
